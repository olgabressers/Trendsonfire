How to relist a delisted article

  1. Remove <meta name="robots" content="noindex, nofollow"/> from the page's HTML
  2. Add the article card back to the relevant section index page
  3. Add the article to the ARTICLES search array on the section index and on index.html
  4. Add the URL back to sitemap.xml
  5. Remove the row from this table and update the Word document
  6. Commit and push to GitHub
